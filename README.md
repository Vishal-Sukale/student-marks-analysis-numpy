# Students Marks Analysis using NumPy (2D Array)

## Problem Statement
Analyzing student performance manually can be time-consuming and error-prone. This project uses NumPy to efficiently analyze student marks.

## Objective
- Analyze student & subject marks  
- Calculate total, average, grades  
- Identify topper  

## Dataset
2D NumPy array (students × subjects)

## Technologies Used
- Python  
- NumPy  

## How to Run
1. Install NumPy  
2. Run notebook  

## Conclusion
NumPy simplifies analysis and improves efficiency.
