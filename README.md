# Students Marks Analysis using NumPy (2D Array)

## Problem Statement
Analyzing student performance manually can be time-consuming and error-prone. This project uses NumPy to efficiently analyze student marks across multiple subjects and generate insights like totals, averages, grades, and topper.

## Objective
- To analyze student-wise and subject-wise marks using a 2D NumPy array  
- To calculate total, average, grades, and result status  
- To identify the topper using efficient array operations  

## Dataset Description
- Data is stored in a **2D NumPy Array**
- Rows represent **students**
- Columns represent **subjects**

### Subjects:
1. Machine Learning  
2. Deep Learning  
3. Mathematics  
4. Science  

### Sample Data:
```
[
 [78, 85, 88, 90],
 [65, 70, 72, 75],
 [92, 90, 95, 94],
 [55, 60, 58, 62],
 [80, 82, 85, 88]
]
```

## Technologies Used
- Python  
- NumPy  

## Operations Performed
- Total elements, shape, dimensions  
- Maximum & Minimum marks  
- Student-wise & subject-wise totals  
- Averages calculation  
- Pass/Fail status  
- Grade assignment (A, B, C, F)  
- Topper identification  

## Key Results
- Highest Marks: 95  
- Lowest Marks: 55  
- Topper identified successfully  
- Automated grading system  

## Key Insights
- NumPy enables fast data processing  
- Easy row-wise and column-wise analysis  
- Reduces manual calculation effort  
- Scalable for large datasets  

## How to Run the Project
1. Install NumPy  
2. Run the notebook or Python file  

## Conclusion
This project demonstrates how NumPy simplifies student performance analysis using efficient array operations. It provides accurate and fast insights with minimal code.
